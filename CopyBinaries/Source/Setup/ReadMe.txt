Content of the file Fragments\SuperDuperLibrary.Binaries.wxs is generated automatically while Bre-built event. 
This file contains dlls that are produced while compiling SuperDuperLibrary.

There is a variable var.SuperDuperLibraryOutputDir in Pre-build event. 
This variable should be defined in preprocessor variable. 
Go to: Setup -> project settings -> Build tab -> Define preprocessor variables.
